parseDateFromString :: String -> Either ParseError TwitterDate #

Tries to parse the given String an instance of TwitterDate. | Notice that thanks to `<* eof` parse is successful only if the whole string meets the predicate, | without it dates like "Thu Oct 27 00:14:27 +0000 201611111" would be cut to "Thu Oct 27 00:14:27 +0000 2016" | without reporting failure
